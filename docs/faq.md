# FAQ

## Does MST install an agent?

No. MST only runs on demand or from cron.

## Does MST require Docker, Prometheus, or Grafana?

No.

## Does MST send telemetry anywhere?

No. The only supported network integration is the Telegram Bot API when enabled.

## Does `mst update` download updates automatically?

No. Automatic self-update is intentionally disabled by security policy.

## Do monitoring commands work yet?

No. The current phase implements only the software foundation. Monitoring commands intentionally return `NOT IMPLEMENTED`.
